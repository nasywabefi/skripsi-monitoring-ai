
Dataset yang digunakan pada penelitian ini terdiri dari 4 video CCTV dari area mesin produksi PT. I Flex Indonesia. Video digunakan untuk pengujian status mesin dan operator pada beberapa kondisi, yaitu:

1. Video malam hari saat mesin tidak aktif
2. Video siang hari saat mesin aktif beroperasi
3. Video saat mesin dibuka/dinyalakan pada pagi hari
4. Video saat mesin dimatikan/ditutup pada malam hari

Folder `sample_output/` berisi file CSV hasil pemrosesan dari empat video uji. File CSV digunakan sebagai data logging hasil monitoring, seperti status pencahayaan, status mesin, nilai movement, nilai EMA, status final mesin, dan status operator.

Dataset video tidak disertakan dalam repository karena merupakan data internal perusahaan. Hasil pengujian dari video tersebut disajikan dalam laporan skripsi.
